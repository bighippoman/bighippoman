#### [Interakly](https://interakly.com) — interactive video platform for education

Upload a video, drop quizzes, polls, and hotspots at any timestamp, share a link. That simple — but the engine underneath is not.

- **25 interaction types** with real-time analytics
- **Live interactive streaming** — WHIP/RTMPS ingest with auto-VOD conversion
- **Sandboxed code editor** running 9 languages via Firecracker microVMs
- **Workspace system** with 10 widget types for building complete learning environments
- **LTI 1.3 integration** for LMS grade passback
- **Electron desktop app** for offline use

Built with Next.js 16, React 19, Convex, Cloudflare Workers (via OpenNext), Cloudflare Stream, Clerk, and E2B.
